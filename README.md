# AGENTIC DEV STUDIO — Website

Landing page for [AGENTIC DEV STUDIO](https://agenticdevstudio.com), the EdTech platform for AI-native developers.

Built with **Expo + TypeScript** targeting web (`expo-router`, `react-native-web`).

## Design

| Token | Value |
|---|---|
| Primary | `#000000` Black |
| Surface | `#111111` Near-black |
| Text | `#FFFFFF` White |
| Muted text | `#9CA3AF` Gray |
| Accent / CTA | `#06D6F0` Cyan |

## Sections

1. **Navbar** — Fixed, blurred, responsive with mobile hamburger
2. **Hero** — Full-viewport with grid background, glow, headline + CTAs + social proof
3. **Stats** — 4 key metrics (2,400+ students, 95% completion, 40+ courses, 100% project-based)
4. **Features** — 6 feature cards in a responsive grid
5. **Courses** — 3 course tracks (Foundations, Multi-Agent Systems, Production AI)
6. **CTA** — Email capture with free access CTA
7. **Footer** — Brand, links, status badge

## Getting Started

```bash
npm install
npm run web       # open http://localhost:8081
```

## Stack

- [Expo SDK 53](https://expo.dev)
- [Expo Router v4](https://expo.github.io/router)
- React Native Web
- TypeScript (strict)
